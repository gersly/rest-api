# REST APIs built with Express JS

## Setup
1. Clone this repository with `git clone git@github.com:gersly/rest-api.git`.
1. Install the dependencies with `npm install `.
1. Run using Node.

## Dependencies 
1. ExpressJS 
1. PostgreSQL
1. Body Parser
1. Sequelize ORM

## messages-api.js
### Routes
1. POST `:3000/messages`
